# Code of Conduct

`Scratch AI Coach` serves both education and open source collaboration. We want this project to stay safe, respectful, and high-trust for learners, teachers, maintainers, and contributors.

## Expected Behavior

- Respect people across age, background, skill level, and communication style
- Keep discussions grounded in facts, reproduction steps, evidence, and improvements
- Be patient with first-time contributors
- Treat classroom, teaching, and youth-facing contexts with extra care

## Unacceptable Behavior

- harassment, threats, discrimination, or personal attacks
- hateful, abusive, or education-inappropriate content
- deliberate disruption, spam, or bad-faith escalation
- publicly dumping exploit details when a security issue is still unpatched

## Enforcement

Maintainers may edit, hide, close, reject, or remove inappropriate contributions. Severe or repeated violations may lead to restricted participation.

## Reporting

- For general community conflicts, de-escalate and return to the facts
- For persistent harassment or unsafe behavior, contact the maintainer
- For vulnerabilities, do not post publicly; see [`SECURITY.en.md`](SECURITY.en.md)

## Scope

This code of conduct applies to issues, pull requests, code review, documentation discussions, and other public interactions directly tied to this project.
