# Roadmap

## Current Phase

The mainline is currently advancing two tracks at the same time:

- maintain stable Windows and macOS support for the local desktop edition
- launch the minimum classroom workflow on the `Python FastAPI + Vue` server track
- keep the repo easy to clone, test, package, and contribute to

## Next Phase

- finish the first server phase for teachers, students, releases, and live dashboards
- improve the GitHub open source collaboration experience
- tighten download, verification, and release documentation

## Future Direction

- support a server + client teaching workflow
- grow a stronger set of classroom examples and verification tooling around Scratch
- expand into courses, classes, history, and richer classroom analytics

## Not in Scope for This Round

- automatic GitHub Releases publishing
- large-scale product internationalization work
- multi-layer course/classroom modeling in the first server phase
- WebSocket/SSE live push in the first server phase
