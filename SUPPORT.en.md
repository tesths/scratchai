# Support

If you want to use, evaluate, or contribute to `Scratch AI Coach`, start with the following entry points.

## Read the Docs First

- project overview: [`README.en.md`](README.en.md)
- repo structure: [`docs/project-structure.en.md`](docs/project-structure.en.md)
- releases and downloads: [`docs/releasing.en.md`](docs/releasing.en.md)
- desktop app details: [`apps/desktop-companion/README.md`](apps/desktop-companion/README.md)

## When to Open an Issue

- Bug: use `Bug Report`
- Feature idea: use `Feature Request`
- Larger direction change: describe the classroom scenario, platform, audience, and expected outcome first

## Helpful Context to Include

- platform: Windows or macOS
- install method: source, portable, installer, zip, or dmg
- whether a DeepSeek key is configured
- reproduction steps
- logs or screenshots

## Security Issues

Do not report them publicly. See [`SECURITY.en.md`](SECURITY.en.md).
